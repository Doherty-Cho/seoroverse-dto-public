### Setup

- frontend setup
  - 각 package.json에 dependencies에 기재되어 별도 setup 불필요
- backend setup

  ```
  // Packages install
  npm install

  // Bundle - dist
  npm run build

  // Link DTO codes
  npm link
  ```

### Typescript .d.ts 파일 생성

```
npm run tsc
```
